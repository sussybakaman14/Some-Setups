# Security Policy

## Supported Versions

Questo repository non contiene software sviluppato da me, ma una raccolta di
installer scaricati nel tempo. Di conseguenza **non esistono versioni del
progetto che richiedono patch di sicurezza**, perché non è un software
manutenuto.

Tuttavia, per chiarezza, ecco la tabella di supporto:

| Version | Supported          |
| ------- | ------------------ |
| Archivio attuale | :white_check_mark: |
| Versioni precedenti | :white_check_mark: |
| Software di terze parti | :x: (non gestito) |

> Nota: Gli installer presenti appartengono ai rispettivi sviluppatori.
> Eventuali vulnerabilità nei programmi inclusi **non possono essere corrette da me**.

---

## Reporting a Vulnerability

Se noti:

- un file sospetto  
- un installer corrotto  
- un link non ufficiale  
- un contenuto che viola copyright o policy  

puoi aprire una **Issue** su GitHub.

### Cosa aspettarti

- Risposta entro 24–72 ore  
- Rimozione immediata di file potenzialmente dannosi  
- Verifica manuale dell’hash o della provenienza  
- Nessuna patch software (non sono l’autore dei programmi)

### Importante

Per vulnerabilità riguardanti i software originali (es. Malwarebytes, Glary Utilities, .NET, ecc.) devi contattare **i rispettivi sviluppatori ufficiali**, non questo repository.
